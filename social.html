📌 本次考試範圍單元

🌾 第1課：農、漁、畜牧業
第一級產業

依自然環境從事生產，滿足日常生活所需或提供其他產業原料。


農業：種植稻米、蔬果、茶葉等農作物
漁業：捕撈（近海、遠洋）或養殖魚、蝦、貝類
畜牧業：飼養雞、鴨、牛、豬、羊等禽畜

📍 靠山吃山、靠海吃海 → 說明居民依自然環境從事生產
① 建造溫室→降低病蟲害、延長採收時程
② 申請產銷履歷→讓消費者安心、建立品牌
③ 品種改良→提升甜度、風味（如愛文/金煌芒果）
④ 遠端監控→溫度、溼度、光照
⚠️ 常考：產銷履歷 = 讓消費者了解來源，不是保證最便宜
養殖漁業：
・遠端監控魚塭水質
・寒流→加熱設施提高水溫，預防魚群凍死

畜牧業：
・培育抗病力較強的品種
・自動投餵飼料、給水裝置
・申請產銷履歷（水產品和畜牧產品也可以申請）
・規畫寬敞自然空間讓雞隻舒適生長
・過去大量砍伐→水土保持和生態問題
・民國80年：政府禁止砍伐天然林
・林場轉型：保育、教育、觀光休閒
・現在：種植人工林、推廣林業文化園區、森林療癒
📝 太平山的「蹦蹦車」：從載木材→載旅客
🏭 第2課：家鄉的工業
第二級產業

使用機器將原物料加工製成食品、衣物、電子產品，滿足食衣住行育樂需求。


・牛奶工廠（食品）
・襪子工廠（衣物）→機器一天可生產好幾百雙
・自行車工廠（零件組裝，人力+機器）
・電子工廠（無塵室、自動化）
・鳳梨酥工廠（食品，電腦管理）
早期：需要較多人力
現在：機器取代人力、遠端監控、降低成本、提高產量

⚠️ 工廠工人需持續學習新技術，否則可能被機器取代

📌 鳳梨酥：手工→電腦管理自動化大量生產
📌 紡織業：人力外移→研發機能布料、回收寶特瓶製環保衣料
🛒 第3課：家鄉的服務業
第三級產業

滿足居民日常生活中食、衣、住、行、育、樂各種需求。


・火車司機（載運乘客、需輪班）
・便利商店店員（販售商品、代收費用）
・記者（採訪、撰寫報導）
・銀行行員（開戶、存提款、金融諮詢）
・圖書館館員、美髮師、外送員等
科技讓服務更多樣：
食：親自到店 OR 網路平台訂購外送到家
衣：自助結帳（感應掃描）
住：科技模擬家具擺放場景
行：手機App搜尋車輛直接租車

美容美髮院 vs 快速剪髮店 → 不同需求選不同服務
社會變遷+生活需求改變 → 產生新興職業：
・網路媒體從業人員（YouTuber）
・線上教學教師
・長照護理師
・寵物保母、寵物美容師、寵物禮儀師
・寵物旅館保母
⚠️ 新興職業出現的原因：社會變遷、生活需求改變
👥 第4課：家鄉人口的分布
第二單元

人口分布不平均，與地形、自然資源、產業、交通有關。


平原、盆地、台地（地勢平坦）：
→ 開發容易、可種農作物、從事工商業
→ 人口多（例：臺中市清水區）

丘陵、山地（地勢起伏）：
→ 開發不易
→ 人口少（例：臺中市和平區）
豐富礦產、森林 → 人口聚集

📍 九份（新北市）：盛產金礦、煤礦 → 1939年人口達1萬8千人 → 停止採礦 → 人口減少 → 現為觀光景點

📍 林田山（花蓮）：檜木 → 全盛期2千多人 → 禁止伐木 → 轉型林業文化園區
工商業發達 + 交通便利 → 人口聚集

📍 臺南市永康區：
有高速公路交流道+火車站 + 鄰近南部科學園區
→ 交通便利、工作機會多 → 人口較多
📊 第5課：家鄉人口的變化
第二單元

人口可分三個年齡階段；人口數因出生、死亡、移入、移出而變化。


幼年人口（0～未成年）
青壯年人口（主要從事生產活動，扶養幼年與老年）
老年人口

透過人口總數變化圖和三階段年齡人口數變化圖了解變化
人口增加：出生、移入
人口減少：死亡、移出

出生受：生育觀念、養育費用影響
死亡受：醫療技術、衛生保健、自然災害影響
移入/移出受：結婚、工作機會、教育環境、交通及公共建設影響
少子化（出生人口少）→
政府推動：生育補助、育兒補助、廣設幼兒園、延長托育

高齡化（老年人口多）→
政府推動：長期照顧政策、社區照顧服務、辦理休閒活動

人口移入過多：交通擁擠、環境汙染、居住空間狹窄
青壯年大量移出：勞動力不足、老年人獨居缺照顧

❓ 容易搞混的地方

A VS B（重要差別）
農業、漁業、畜牧業
（第一級產業）		 ≠ 工業（第二級）是把原料加工製造成產品，不是直接採集自然資源
捕撈漁業 ≠ 養殖漁業：養殖是在魚塭裡養，可用科技（遠端監控、加熱設施）
產銷履歷
= 讓消費者安心了解來源		 ≠ 不是保證最便宜，也不是保證最好吃，是透明化農產品資訊
寒流來時（漁業）
用加熱設施提高水溫		 ≠ 颱風時漁業不是「加熱」，而是加蓋魚塭；寒流才是加熱！
少子化
＝出生人口少		 ≠ 高齡化＝老年人口比例高。兩者可以同時發生，臺灣就是這樣
移入人口增加
→ 人口成長		 ≠ 移出人口增加→ 人口減少；青壯年移出會造成勞動力不足
地形平坦
（平原、盆地、台地）
→ 人口多		 ≠ 地形起伏（丘陵、山地）→ 人口少（開發困難）
但若有豐富礦產，山地也可能聚集人口（如九份）
九份（採礦時）
→ 人口多（礦業興盛）		 → 九份（現在）→ 人口少、轉型觀光；礦業停止後人口外移
服務業的特色 ≠ 服務業提供「服務」不是有形「產品」；依需求可選不同服務方式（如美髮院vs快速剪髮）
新興職業出現的原因 → 社會變遷 + 生活需求改變（例如寵物保母因寵物視為家人而出現）

⚡ 考前 5 分鐘速覽

🌾 農漁畜牧（第一級）

  • 農業 = 種作物；漁業 = 捕/養；畜牧 = 飼養禽畜
  • 「靠山吃山、靠海吃海」
  • 溫室 → 降低病蟲害
  • 產銷履歷 → 讓消費者了解來源
  • 寒流 → 加熱設施提高水溫
  • 林業：民國80年禁砍天然林

🏭 工業（第二級）

  • 原料 → 加工 → 食品/衣物/電子產品
  • 現在：機器取代人力、遠端監控
  • 工人要持續學習新技術
  • 電子工廠 → 無塵室
  • 鳳梨酥：手工→自動化
  • 紡織業：寶特瓶→環保布料

🛒 服務業（第三級）

  • 提供服務，滿足食衣住行育樂需求
  • 科技讓服務多樣化（外送、自助結帳）
  • 新興職業：YouTuber、線上教師、寵物保母
  • 原因：社會變遷、生活需求改變

👥 人口分布

  • 平原→人多；山地→人少
  • 自然資源豐富→人口聚集
  • 九份（採礦→觀光）
  • 工商業發達+交通便利→人口多
  • 臺南永康：高速公路+科學園區

📈 人口變化原因

  • 增加：出生、移入
  • 減少：死亡、移出
  • 三階段：幼年、青壯年（主生產）、老年
  • 移入移出受：結婚、工作、教育、交通影響

⚠️ 人口問題與對策

  • 少子化→生育補助、廣設幼兒園
  • 高齡化→長期照顧、社區照顧
  • 人口過多→交通擁擠、環境污染
  • 青壯年移出→勞動力不足
